# 🕵️ InspectorAI Bot

![Python Version](https://img.shields.io/badge/python-3.10%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

**InspectorAI** — это многофункциональный Telegram-бот, созданный для улучшения различных аспектов вашей жизни. Он сочетает в себе мощь передовых языковых моделей (LLM), инструменты для отслеживания фитнес-прогресса, персонального диетолога и удобный менеджер для учета групповых финансов.

---

## 🚀 Ключевые возможности

Бот состоит из нескольких независимых модулей, доступных из главного меню:

### 🏋️ Тренер
Персональный помощник для отслеживания и планирования тренировок.

*   **Создание шаблонов:** Легко создавайте и настраивайте свои программы тренировок через пошаговый диалог.
*   **Выполнение тренировок:** Запускайте тренировку в интерактивном режиме. Бот будет вести вас от упражнения к упражнению, от подхода к подходу.
*   **Интерактивный логгинг:** Удобно изменяйте вес и количество повторений с помощью кнопок прямо во время выполнения подхода.
*   **История и прогресс:** Бот запоминает ваши предыдущие результаты и показывает их при выполнении упражнения, чтобы мотивировать на прогресс.
*   **Статистика и анализ:** Отслеживайте свой прогресс в конкретных упражнениях, просматривайте динамику весов и 1ПМ (одноповторного максимума).
*   **Анализ от ИИ:** Получите развернутый анализ вашего прогресса и рекомендации от ИИ-тренера на основе ваших данных.

### 🥗 Нутрициолог
Ваш карманный диетолог для контроля за питанием.

*   **Персональный план:** Бот рассчитывает вашу индивидуальную норму калорий, белков, жиров и углеводов (КБЖУ) на основе ваших параметров (рост, вес, возраст, активность) и целей (похудение, набор массы, рекомпозиция).
*   **Логирование еды по фото:** Просто сфотографируйте свою еду! Мультимодальная ИИ-модель распознает блюда и автоматически рассчитает их КБЖУ.
*   **Статус в реальном времени:** Наглядный дашборд с прогресс-барами показывает, сколько нутриентов вы уже потребили за день и сколько осталось до цели.
*   **"Умные" рецепты:** Если вы не знаете, что съесть, чтобы уложиться в дневную норму, бот проанализирует ваш рацион и предложит конкретные варианты блюд.
*   **Статистика:** Просматривайте сводку и средние показатели вашего рациона за последнюю неделю.

### 💸 Финансы (Модуль "Ланистер")
Инструмент для удобного управления общими расходами в групповых чатах.

*   **Учет расходов:** Быстрая запись трат прямо в чате.
*   **Умный подсчет:** Автоматический расчет долей и взаимозачет встречных долгов для минимизации количества транзакций.
*   **Прозрачность:** Команда `баланс` в любой момент покажет, кто кому и сколько должен.
*   **"Час расплаты":** Интерактивный процесс погашения долгов с подтверждением от всех участников.

### 🧠 Инспектор (ИИ-Ассистент)
Универсальный помощник на базе LLM.

*   **Мультимодельность:** Возможность переключаться между `Gemini 1.5 Flash` и различными моделями через `OpenRouter`.
*   **Анализ контента:** Бот может анализировать информацию, проводить фактчекинг и отвечать на вопросы в контексте диалога.
*   **Транскрибация:** Автоматическое преобразование голосовых сообщений в текст.
*   **Исправление ссылок:** Корректирует ссылки на популярные соцсети для прямого доступа к контенту.

## 🛠 Технологии

*   **Язык:** [Python 3.10+](https://www.python.org/)
*   **Фреймворк:** [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) (Async)
*   **Базы данных:** SQLite для модулей тренировок и питания.
*   **AI Provider:** Google Gemini API / OpenRouter API
*   **Тестирование:** Pytest
*   **CI/CD:** GitHub Actions

## 📦 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/PreparingShaders/InspectorAI.git
cd InspectorAI
```
### 2. Создание и активация виртуального окружения
Рекомендуется использовать виртуальное окружение:
```bash
python -m venv venv
```
**Windows:**
```bash
venv\Scripts\activate
```
**Linux/MacOS:**
```bash
source venv/bin/activate
```
### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 4. Запуск бота
```bash
python main.py
```

## 🧪 Тестирование
Проект частично покрыт юнит-тестами. Тесты запускаются автоматически при каждом пуше в репозиторий через GitHub Actions.

**Запуск локально:**
```bash
pytest -v
```
